Cientista de Dados

Atuando em posições estratégicas realizando análises utilizando ferramentas de análise de dados como R, RStudio, Python e ferramentas de visualização de dados como ggplot2, Tableau dentre outras.

Dentre as metodologias estatísticas, utilize: teste A/B, análise de regressão, algoritmos de segmentação/agrupamento, modelos de classificação como regressão logística, modelos de inferência causal como propensity score matching, dentre outras.
